# ステージ仕様書

## ステージ形式の種類

| 形式 | 説明 |
|------|------|
| 横移動形式 | プレイヤーが右へ進み、Gemに触れるとクリア。敵はステージ内に固定配置または Spawner で管理される |
| ウェーブ形式 | プレイヤーが特定エリアに留まり、WaveManager が時間差で敵を出現させる。全波が出し切られた後、一定時間でGemが出現する |

---

### Stage 5

| 項目 | 内容 |
|------|------|
| 形式 | ウェーブ形式 |
| 敵の配置方式 | WaveManager によるコルーチン管理 |
| 登場する敵 | Frog × 8、Eagle × 4 |

#### WaveManager の動作仕様

- 各ウェーブは「出現時間・出現位置・存在時間」を個別に設定できる
- ステージ開始と同時に全ウェーブのコルーチンが起動し、設定時間になると敵が出現する
- 各敵は設定された存在時間が経過するか、プレイヤーに倒されると消滅する
- 全ウェーブが出現し終えてから22秒後にGemが出現し、クリア可能になる

---

Spawner の詳細な動作は [enemies.md](./enemies.md) を参照。
