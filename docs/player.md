# プレイヤー仕様書

## 操作

| 操作 | キー |
|------|------|
| 左移動 | `A` キー または 左矢印キー |
| 右移動 | `D` キー または 右矢印キー |
| ジャンプ | `Space` キー |

## 移動

- 左右キーの入力に応じて水平方向に力を加えて加速する
- 入力がない場合は自然に減速して停止する
- 移動方向に応じてスプライトが左右に反転する
- 移動速度に応じて走りアニメーションが再生される（速度0で停止アニメ）

## ジャンプ

- 地面または坂道に接地している状態でのみジャンプできる
- ジャンプ中はジャンプアニメーションが再生される
- 上昇から下降に転じるとフォールアニメーションに切り替わる

## 地面・坂道判定

- コライダーの前後2点から下向きにレイキャストを飛ばして接地判定を行う
- 前後どちらか片方のみヒットした場合は「坂道上」と判定する
- 坂道上で静止している場合はY方向の移動をフリーズし、ずり落ちを防止する
- 坂道を登りきった直後は一瞬下方向に速度を与えて地面への着地を補助する

## 踏みつけ（敵を倒す）

- 敵の位置がプレイヤーの足元より下にある場合、踏みつけとして判定する
- 踏みつけた敵はその場で消滅する
- 踏みつけ後、プレイヤーは自動的に上方向へバウンスする

## ダメージ（ゲームオーバー）

- 踏みつけ以外の角度で敵に接触した場合、即座にゲームオーバーになる

## ゴール

- `Gem` レイヤーのオブジェクトに触れるとステージクリアになる

## 落下死

- Y座標が一定値を下回ると落下死として即座にゲームオーバーになる
- 落下死は一度だけ処理される（重複実行を防止するフラグあり）
